A Template Toolkit plugin to display audit events recorded by [Audit::DBI]
(https://metacpan.org/release/Audit-DBI).


INSTALLATION
------------

To install this module, run the following commands:

	perl Build.PL
	./Build
	./Build test
	./Build install
